Tech Quiz Application Overview

Java Backend

Core logic, score calculation and server-side processing using Java.

JSP Pages

Dynamic web pages for login, exam, result and admin modules.

Apache Tomcat

Server used to deploy and run the Tech Quiz application.

WAMP / MySQL

Database used to store users, questions, results and login data.

HTML, CSS, JS

Frontend design, validation, animations and responsive layout.

User Module

Users can register, login, attend quiz and view results instantly.

Login System

Secure admin and user login authentication system.

Admin Panel

Admin can add questions, manage users and monitor quiz results.

Instant Results

Automatic score calculation and performance evaluation.
